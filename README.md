
# 北海学園大学一部新聞会 公式サイト (Production)

このプロジェクトは、北海学園大学一部新聞会の公式ニュース配信サイトです。

## プロジェクトの役割

- **フロントエンド（本サイト）**: 閲覧専用。記事の表示、広告の表示、メンテナンス画面の切り替えなどを行います。
- **管理サイト（別プロジェクト）**: コンテンツの作成、更新、削除、noteとの同期、サイト設定の変更を行います。

## 接続情報

本サイトは以下の Firebase プロジェクトを参照しています。管理サイトも同じプロジェクトを参照するように設定してください。

- **Project ID**: `studio-7293379319-74783`

## 運用・管理

管理用アプリからデータを更新すると、本サイトは Firestore のリアルタイム同期機能により即座に表示が更新されます。
