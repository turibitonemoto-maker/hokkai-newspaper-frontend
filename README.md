
# 北海学園大学一部新聞会 公式サイト (表示用サイト)

このプロジェクトは、北海学園大学一部新聞会の公式ニュース配信における**閲覧専用（表示用）サイト**です。

## プロジェクトの役割

- **こちら表示用サイト**: 閲覧専用のフロントエンドです。管理サイト（別プロジェクト）で更新されたFirestoreのデータをリアルタイムに取得し、読者に表示します。
- **データ連動**:
    - `isPublished: true` の記事のみを表示します。
    - `settings/maintenance` ドキュメントを監視し、メンテナンスモードの切り替えを即座に反映します。
- **管理機能の排除**: 誤操作やセキュリティリスクを避けるため、本プロジェクトには記事の投稿や削除などの書き込み機能、および管理者ログイン画面は一切含まれていません。

## 技術的特筆事項

- **Firebase SDK シングルトン管理**: Next.jsのホットリロード時におけるFirestoreの内部エラー（ID: ca9）を防止するため、`globalThis` を利用した物理的なインスタンス固定を行っています。
- **タイポグラフィ**: 1950年創立の伝統を表現するため、重厚な明朝体（Yuji Mai）と現代的なサンセリフを組み合わせた独自のスタイルを採用しています。
